# 项目接手文档

## 阅读入口

- [开发文档索引](./开发文档索引.md) — 所有文档的导航入口，按用途分组

## 30 分钟上手路径

1. [项目控制台](./项目控制台.md) — 了解当前阶段和活动 backlog
2. [当前实现总览](./当前实现总览.md) — 已实现能力地图
3. [内部调用链参考](./内部调用链参考.md) — 核心业务调用链（含 Mermaid 时序图）
4. [模块总览](./modules/00-模块总览.md) — 各模块职责、关键文件和接口
5. [运维排障手册](./运维排障手册.md) — 常见问题排查与恢复命令

## 文档维护原则

- 文档与代码同仓维护。
- 文档中的代码位置需标注 `file_path:line_number`。
- 功能行为变更时同步更新对应模块文档和 backlog。
