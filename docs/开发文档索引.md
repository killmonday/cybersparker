## 文档维护触发点

修改以下路径时需同步更新对应文档：

| 源码路径 | 需更新的文档 |
|---|---|
| `cybersparker/urls.py` | 对应模块文档 |
| `app_cybersparker/models.py` | 对应模块文档 |
| `app_cybersparker/views/**` | 对应模块文档、调用链参考 |
| `app_cybersparker/templates/**` | 对应模块文档 |
| `app_cybersparker/services/**` | 对应模块文档 |
| 功能行为变更 | `CHANGELOG.md`、模块文档、backlog |
| 阶段/backlog 状态变更 | `项目控制台.md` |

> 过去存在编号前缀的文档（`00_scope_and_conventions.md` 等），它们的内容已合并到模块文档或项目控制台中，2026-05-25 已清理。勿再新建此类编号前缀的扁平文档。
