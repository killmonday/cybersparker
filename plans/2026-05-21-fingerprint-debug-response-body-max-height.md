# 2026-05-21 指纹调试页响应正文高度限制

## 做什么
- 限制指纹调试页“响应正文”展示框的最大高度。
- 当 HTML 响应很长时，正文区域内部滚动，不再把整个页面撑得很长。

## 为什么
- 当前长 HTML 会把正文卡片直接撑高，导致整页过长，查看其它信息不方便。

## 怎么做
1. 只改 `app_cybersparker/templates/project/expload/fingerprint_debug.html`。
2. 给响应正文展示区增加最大高度和内部滚动。
3. 同步 `CHANGELOG.md`；模块文档和 backlog 不改，因为这次只是本地展示体验微调，不改接口和模块范围。

## 风险
- 只影响调试页正文展示样式，风险低。

## 验证
- `python manage.py check`
- 如条件允许，启动页面人工确认滚动行为。

## 当前状态
- 已完成样式调整，准备做最小自检。

## 结果
- 给 `#response-content` 增加了最大高度限制。
- 长 HTML 现在会在正文框内部滚动，不再把整页拉得过长。
- 只改了模板样式，没有改接口和匹配逻辑。

## 后续
