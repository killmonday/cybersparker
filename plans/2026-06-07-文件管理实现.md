# 文件管理功能实现

## 做什么

BL-FILE-001: 新建文件管理页（React + 后端 API），BL-FILE-002: Celery Beat 定时清理引擎数据。

## 怎么做

1. 后端 views：`app_cybersparker/views/expload/target_file_manage.py`（列表/上传/下载/删除/批量删除）
2. URL 路由：`cybersparker/urls.py` 新增 5 条路由
3. React 页面：`frontend/src/pages/TargetFileListPage.tsx`
4. 侧边栏 + 路由：`SidebarLayout.tsx` + `App.tsx`
5. Celery 定时任务：`tasks.py` + `settings.py` Beat schedule

## 风险

- 路径穿越：双重校验（`get_absolute_target_path` + EXP_input 目录边界）
- 并发重名：循环检查文件系统直到不冲突
- 删除引用清理：同步清理 `history_files` 逗号分隔字段
