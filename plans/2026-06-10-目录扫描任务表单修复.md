# 目录扫描任务表单 — 代理下拉修复 + 源任务翻页美化

## 问题1：代理地址下拉空白

**根因**：前端 `ProxyOption` 接口定义为 `{ id, name, host }`，但后端 `/api/v1/proxies` API 返回 `{ id, proxy_type_label, proxy_address, proxy_port }`。ProxySetting 模型根本没有 `name` 字段。渲染时 `p.name` 和 `p.host` 都是 `undefined`，所以下拉选项文字为空。

**修复**：把 `ProxyOption` 接口改为匹配 API 实际返回字段，渲染时用 `proxy_type_label://proxy_address:proxy_port` 显示。

## 问题2：源任务无翻页、样式简陋

**修复**：
- 源任务区域加搜索框，支持按任务名搜索
- 分页加载（每页10条），加"加载更多"翻页逻辑
- 容器美化：选中项染色、hover 效果、搜索高亮、全选/取消全选按钮
