# 2026-06-14-BL-AIPOC-010-任务列表翻页

## 做什么

给 AI 生成 PoC 任务列表（`PocGenTaskListPage.tsx`）加翻页功能，参考 `DictListPage` 的翻页模式。

## 怎么做

### 后端（1 文件）
- `poc_gen_task.py` `api_tasks` GET：引入 `Pagination`，接收 `page`/`rows_per_page` 参数，返回分页数据 + `page`/`total_pages`/`rows_per_page`

### 前端（2 文件）
- `types/pocGenTask.ts`：`PocGenTaskListResponse` 加 `page`/`total_pages`/`rows_per_page`
- `PocGenTaskListPage.tsx`：加 `page` state、翻页 bar（模仿 DictListPage）、`load()` 传分页参数

## 风险
- 低：改动局限在一个 API + 一个页面，无跨模块影响

## 验证
- tsc 编译通过 ✓
- Django check 0 错误 ✓

## 结果
翻页功能已实现。API 默认每页 10 条，前端与 DictListPage 风格一致的翻页栏。
